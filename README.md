# EV Charging Dashboard

## Overview

This project presents an interactive Power BI dashboard for analyzing Electric Vehicle (EV) charging station data. The dashboard provides insights into charging trends, station performance, vehicle activity, electricity load, weather conditions, and waiting times.

## Features

* Interactive KPIs to monitor key metrics.
* Dynamic charts and graphs for data visualization.
* Slicers and filters for customized analysis.
* Analysis of charging types, vehicle activity, and station utilization.
* User-friendly dashboard for data-driven decision-making.

## Tools Used

* Power BI
* Power Query
* DAX
* EV Charging Dataset

## Dashboard Insights

* Charging station performance
* Vehicle charging trends
* Waiting time analysis
* Electricity load monitoring
* Weather impact on charging activity
* Charging type distribution

## Conclusion

This dashboard demonstrates business intelligence and data visualization techniques using Power BI, enabling users to explore EV charging data through interactive reports and meaningful insights.
